## ✨ About

Florist Studio is an installable Progressive Web App (PWA) built specifically for florists.

It replaces multiple spreadsheets, notebooks and apps with one beautiful workspace designed for weddings, daily orders, customer management, pricing, loyalty, inventory and business administration.
