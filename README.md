# 🌸 Florist Studio

An all-in-one florist business dashboard created by **Eternal Blooms**.

Florist Studio helps florists price bouquets, save orders, track expenses, manage inventory, create invoices, manage customers, view analytics and generate content ideas — all in one place.

## Features

- Bouquet pricing calculator
- Inventory management
- Calculator-to-order flow
- Customer management
- Downloadable invoices
- Expenses tracker
- Calendar/order diary
- Analytics and downloadable charts
- Content ideas page
- Local-first data with export/import
- Mobile-friendly PWA layout

## Privacy

Florist Studio is local-first. It does not collect user data. Each florist keeps their business information on their own device/browser and can export backups.
